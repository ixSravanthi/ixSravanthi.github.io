Projects

EfficientNet and DWT-Driven Intelligent Skin Lesion Classification

Developed an intelligent skin lesion classification system that combines Discrete Wavelet Transform (DWT) based feature decomposition with EfficientNet to improve classification performance on dermoscopic images. The approach leverages multi-resolution texture information from DWT.

Methodology

  • Applied Discrete Wavelet Transform (DWT) for multi-resolution feature extraction
  • Preprocessed dermoscopic images to enhance lesion features
  • Used EfficientNet as the backbone CNN for classification
  • Integrated frequency-domain and spatial features
  • Trained and evaluated the model on labeled skin lesion datasets
